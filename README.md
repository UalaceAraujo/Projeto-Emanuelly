# 💄 Emanuely Designer de Sobrancelhas  
### Projeto de Extensão Universitária

Sistema web de agendamento online desenvolvido como **Projeto de Extensão da Faculdade**, com o objetivo de aplicar conhecimentos acadêmicos em um projeto real para uma profissional da área da beleza.

---

## 📚 Sobre o Projeto

Este projeto foi idealizado para atender a empresa **Emanuely Designer de Sobrancelhas**, oferecendo uma solução digital para organização de agendamentos e apresentação profissional da marca.

A proposta integrou teoria e prática, aplicando conceitos de:

- Desenvolvimento Web
- Arquitetura de Software
- Organização de Código
- Experiência do Usuário (UI/UX)
- Estruturação de sistemas escaláveis

---

## 🎯 Objetivos Acadêmicos

- Aplicar conhecimentos técnicos em um cenário real
- Desenvolver uma aplicação funcional e responsiva
- Trabalhar boas práticas de desenvolvimento
- Organizar estrutura de frontend e backend
- Simular ambiente profissional de entrega de software

---

## 🚀 Funcionalidades

- 📅 Sistema de agendamento online
- 💅 Exibição de serviços oferecidos
- 📱 Layout responsivo (Desktop e Mobile)
- 🧾 Organização de horários
- 🏗 Estrutura preparada para expansão futura

---

## 🛠 Tecnologias Utilizadas

### Frontend
- React.js
- Next.js
- HTML5
- CSS3
- JavaScript

### Backend
- API para gerenciamento de agendamentos
- Estrutura para integração com banco de dados

### Banco de Dados
- Sistema de armazenamento para controle de horários
